# 🖐️ Gesture to Meme

Real-time hand gesture recognition that displays memes when you make specific gestures!

## ✨ Features

- 🎯 4 gestures recognized (Thumbs up, Peace, Pointing, Open palm)
- 👐 Multiple hands supported simultaneously
- 🎲 Random meme selection per gesture
- 📷 Real-time hand tracking (21 landmarks)
- 🚀 Easy setup - just double-click!

## 🚀 Quick Start

### Windows Users (Easiest)
1. Download and extract the ZIP
2. Double-click `setup_and_run.bat`
3. If Python is missing, install from the link
4. Show hand gestures to your camera!

### Manual Setup
